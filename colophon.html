Colophon

The type­face is ET Book.

I write this site us­ing Emacs and DrRacket in a cus­tom markup lan­guage that is then ren­dered into HTML and CSS us­ing the Pollen pub­lish­ing pro­gram. The source for this site is here.

I have been heav­i­ly influenced by Matthew Butterick’s Practical Typography and I copied many CSS tech­niques (e.g. un­der­lin­ing, re­spon­sive mar­gin notes) from Tufte CSS.
